# Arumugam - AWS & DevOps Engineer Portfolio

A modern, creative, and professional portfolio website showcasing AWS and DevOps engineering expertise, projects, and technical skills.

##  Design Features

- **Dark Theme with Colorful Accents**: Modern dark background (#0a0e27) with vibrant cyan (#00d4ff), purple (#8000ff), and pink (#ff006e) gradients
- **Creative & Professional**: Combines stunning visual effects with professional content presentation
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Floating cards, blob animations, and scroll-triggered reveals
- **Interactive Elements**: Hover effects, smooth transitions, and engaging UI components

##  File Structure

### 1. **Navigation Bar**
- Fixed sticky navigation with smooth scroll
- Active link indicator
- Responsive hamburger menu for mobile

### 2. **Hero Section**
- Eye-catching introduction
- Call-to-action buttons
- Floating animated cards showcasing key skills
- Animated blob backgrounds for visual interest

### 3. **About Section**
- Professional introduction
- Key statistics (Years of Experience, Projects Completed, Technologies Mastered)
- Tech stack visual display

### 4. **Projects Section**
- Three featured projects with detailed descriptions
- Technology tags for each project
- Hover effects and interactive overlays
- Responsive grid layout

  **Projects Included:**
  - AWS Multi-Region High-Availability & Disaster Recovery Architecture
  - Containerized Java Deployment Pipeline
  - AI Chat Bot Infrastructure

### 5. **Skills Section**
- Organized by categories
- Animated skill progress bars
- Visual skill levels
- Categories: Cloud & Infrastructure, Containerization & Orchestration, CI/CD & DevOps, Monitoring & Observability, System Administration, Version Control

### 6. **Contact Section**
- Contact information cards
- Social media links
- Functional contact form
- Professional engagement options

### 7. **Footer**
- Copyright information
- Social media links
- Professional branding
