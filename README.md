# Arumugam - AWS & DevOps Engineer Portfolio

A modern, creative, and professional portfolio website showcasing AWS and DevOps engineering expertise, projects, and technical skills.

##  Design Features

- **Dark Theme with Colorful Accents**: Modern dark background (#0a0e27) with vibrant cyan (#00d4ff), purple (#8000ff), and pink (#ff006e) gradients
- **Creative & Professional**: Combines stunning visual effects with professional content presentation
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Floating cards, blob animations, and scroll-triggered reveals
- **Interactive Elements**: Hover effects, smooth transitions, and engaging UI components
